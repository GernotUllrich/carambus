Run automated grep/analysis commands against all 72 test files to collect issue data per STANDARDS.md categories. Execute these scans in sequence:

**Scan 1: Skipped/pending tests (E03)**
```bash
grep -rn "skip\b\|pending\b" test/ --include="*_test.rb" | grep -v "skip_update\|skip_callbacks\|skipSelf\|skipping"
```
Record file, line number, and skip reason for each match.

**Scan 2: Empty tests / missing assertions (E01, E04)**
For each test file, count assertion calls per test method. A test method with zero `assert` or `refute` calls is E01 (empty) or E04 (missing assertion).
```bash
# Find test methods and count assertions in each
for f in $(find test -name "*_test.rb" -type f | sort); do
  echo "=== $f ==="
  # Count test methods
  grep -c 'test "\|def test_' "$f"
  # Count assertions
  grep -c 'assert\|refute\|must_\|wont_\|should\b' "$f"
done
```
Flag files where assertion count is suspiciously low relative to test count (ratio < 1.0 means some tests have no assertions).

**Scan 3: Weak assertions (E02)**
```bash
grep -rn "assert true\b\|assert false\b" test/ --include="*_test.rb"
grep -rn "assert_not_nil.*[^,]$" test/ --include="*_test.rb" | head -30
```
Note: `assert_not_nil` alone is only weak if it's the ONLY assertion in a test. Cross-reference with per-method analysis.

**Scan 4: Naming violations (W01)**
```bash
grep -rn "def test_" test/ --include="*_test.rb" | grep -v "test_helper\|test_order"
```
Count `def test_` occurrences per file. Files with ALL methods using `def test_` are higher priority than files with mixed usage.

**Scan 5: Setup pattern deviations (W02)**
```bash
# FactoryBot usage
grep -rn "build(\|create(\|build_stubbed(" test/ --include="*_test.rb" | grep -v "Model\.\|\.create\|Hash\[" | head -30
# Direct Model.create! usage (informational, not a violation)
grep -rn "\.create!" test/ --include="*_test.rb" | wc -l
```

**Scan 6: Test method count per file**
```bash
for f in $(find test -name "*_test.rb" -type f | sort); do
  count=$(grep -c 'test "\|def test_' "$f" 2>/dev/null || echo 0)
  lines=$(wc -l < "$f")
  echo "$f: $count tests, $lines lines"
done
```
