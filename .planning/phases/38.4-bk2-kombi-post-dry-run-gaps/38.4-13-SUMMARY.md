## Decisions Made

- **Stub mechanic for T-P1-fallback-when-discipline-missing:** Used `Discipline.method(:find_by)` to capture the real method, wrapped in a `->(arg)` lambda that returns nil only for the `BK100` lookup. This `Discipline.stub :find_by, stub do … end` pattern works correctly in `ActionDispatch::IntegrationTest` because the stub propagates into the controller request thread. No need for the alternative `Discipline.where(name: "BK100").destroy_all + restore-via-teardown` approach. Verified by passing test execution.
- **BK2-Kombi (id 107) test fixture:** The `bk2_kombi:` fixture in `test/fixtures/disciplines.yml` already carries `data: '{"free_game_form":"bk2_kombi","ballziel_choices":[50,60,70]}'`. T-P3 therefore exercises the live `Discipline.find_by` → `discipline.ballziel_choices` path, NOT the new fallback constant. The fallback path for T-P3 in production (where BK2-Kombi seed has not yet been re-run with [50,60,70]) is a defense-in-depth guard, but the controller test does not exercise it directly. T-P1-fallback-when-discipline-missing is the test that exercises the fallback constant directly.
- **Slice-list precondition (I-13-01) confirmed:** `:balls_goal_a` and `:balls_goal_b` are present in the controller's `permit/.slice` list at line 88 (was 74 in the plan reference, drifted to 88 due to comment additions). The Step 0 verification was a sanity check, not a fix — no slice-list edit needed.
- **T-O1-balls-goal-honored regex update:** Plan 38.4-10's file-grep guard was a Plan-10 invariant. Plan 38.4-13 legitimately widens the read pattern, so the regex needed updating from `requested = p\.delete\(:balls_goal\)\.to_i` to `requested = \(?p\.delete\(:balls_goal\)\.presence` (allows leading paren or none). The underlying invariant — `balls_goal` flows from `p`, intersects with `allowed`, defaults to `allowed.first` — is preserved. This counts as Rule 3 (auto-fix blocking issue) since failing to update T-O1 would have left `bin/rails test test/system/bk2_scoreboard_test.rb` with 1 net new failure caused by Plan 13.
