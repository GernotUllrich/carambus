# Phase 38.3 Plan 03: Merge BK2 Scoreboard Partials into Shared Karambol Partials Summary

**One-liner:** Deleted `_scoreboard_bk2_kombi.html.erb` and merged BK2-Kombi rendering into `_scoreboard.html.erb` (phase chip + remaining badge replacing inning counter) and `_player_score_panel.html.erb` (set-score display, GD/HS hidden, negative-tolerant).

## What Was Built

Two tasks merged the separate BK2-Kombi scoreboard partial into the shared Karambol partials via conditional branches on `options[:free_game_form] == "bk2_kombi"`:

### Task 1 — `_player_score_panel.html.erb` BK2 branch

Added `is_bk2` local and derived BK2 state reads at the top of the Ruby header block:
- `bk2_set_score` — current set score for this player from `bk2_state["set_scores"]`
- `bk2_inning_running` — live Aufnahme running total from `innings_redo_list[-1].to_i` (0 when not active)
- `bk2_display_score` — sum of the two (may be negative, renders naturally)
- `bk2_set_target` — from `bk2_state["set_target_points"]`

Five branch sites modified:
1. **Left Ziel slot** — shows `bk2_set_target` instead of `balls_goal` (D-08)
2. **Right Ziel slot** — mirror (D-08)
3. **Left GD/HS rows** — hidden via `|| is_bk2` guard (D-08)
4. **Right GD/HS rows** — hidden via `|| is_bk2` guard (D-08)
5. **Left inning-score span** — shows `bk2_inning_running` for BK2 (D-07)
6. **Right inning-score span** — shows `bk2_inning_running` for BK2 (D-07)
7. **Large center-score** — `score_data_value` local + `bk2_display_score` branch (D-06)

Karambol, Eurokegel, Pool, Biathlon paths unchanged.

### Task 2 — `_scoreboard.html.erb` center strip + deletion

Added BK2 state locals block after the color-class locals (Step A):
- `bk2_current_phase`, `bk2_shots_left`, `bk2_innings_left`
- `bk2_phase_label` — with `present?` guard (GAP-03 port — prevents Rails rendering parent hash)
- `bk2_phase_bg` — amber for DZ, sky for SP, gray for unset

Branched the center strip (h-2/6 div, Step B):
- **BK2 path**: phase chip (colored pill) + remaining badge (shots/innings left) + `bk2-kombi-scoreboard` CSS class
- **Karambol path**: unchanged inning counter + "of N" line

Deleted `_scoreboard_bk2_kombi.html.erb` via `git rm`. Only comment references remain (discipline.rb:148 — documentation trail, not a render call).
