# Phase 38.3 Plan 05: BK2 Shootout 4-Button first_set_mode Matrix Summary

**One-liner:** Branched `_shootout.html.erb` to render a 2×2 grid of 4 Anstoß×first_set_mode buttons for BK2-Kombi matches, with CLAMPED `bk2_first_set_mode` dataset attr persisted into `bk2_options` via `start_game`/`switch_players_and_start_game` reflex augmentations.

## What Was Built

### Task 1 — Branch shootout view + augment start_game reflexes

**Step A — Reflex augmentation (`app/reflexes/table_monitor_reflex.rb`)**

Added identical BK2 first_set_mode block to the top of both `start_game` and `switch_players_and_start_game` (right after the `TableMonitor.find` call):

```ruby
# Phase 38.3-05 D-18: if this TableMonitor is BK2-Kombi and the shootout button
# carried a bk2_first_set_mode, persist it into bk2_options before transitioning.
if @table_monitor.data["free_game_form"] == "bk2_kombi"
  bk2_mode = element.andand.dataset[:bk2_first_set_mode].to_s
  if %w[direkter_zweikampf serienspiel].include?(bk2_mode)
    @table_monitor.data["bk2_options"] ||= {}
    @table_monitor.data["bk2_options"]["first_set_mode"] = bk2_mode
  end
end
```

Security properties:
- T-38.3-05-01 (Tampering): CLAMP whitelist — only `direkter_zweikampf` or `serienspiel` are written; any other DOM value is silently dropped
- T-38.3-05-02 (Tampering): `free_game_form` check is server-controlled JSONB — Karambol games cannot be forced into this branch

**Step B — Shootout view branch (`app/views/table_monitors/_shootout.html.erb`)**

Lines 1-28 (locals + player name header) unchanged. Replaced lines 29-41 (old 2-button block) with:

```
is_bk2 = options[:free_game_form] == "bk2_kombi"
if is_bk2:
  → 2×2 grid: amber DZ buttons (A+B) + sky SP buttons (A+B)
  → A buttons → TableMonitorReflex#start_game + data-bk2-first-set-mode
  → B buttons → TableMonitorReflex#switch_players_and_start_game + data-bk2-first-set-mode
  → Legend: t("table_monitor.bk2_kombi.first_set_mode.legend")
else:
  → Original 2 Karambol buttons (left_player stößt an / right_player stößt an)
```
