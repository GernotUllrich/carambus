# Phase 38.3 Plan 02: Collapse BK2-Kombi into shared _show.html.erb Summary

**One-liner:** Deleted `_show_bk2_kombi.html.erb` and routed BK2-Kombi through `_show.html.erb` by porting the GAP-05 fallback banner as an if/else branch safe for non-BK2 disciplines.

## What Was Built

Two tasks collapsed the separate BK2-Kombi top-level show partial back into the shared Karambol `_show.html.erb`:

1. **Task 1** — Added a `bk2_state_uninitialized?` guarded fallback-banner block to `_show.html.erb`, inserted between the warning-modal block and the existing scoreboard dispatch. The `bk2_state_uninitialized?` predicate (added in Plan 38.2-01) returns `false` for non-BK2 games, making this branch a no-op for Karambol and Eurokegel.

2. **Task 2** — Removed the `elsif free_game_form == "bk2_kombi"` branch from `_table_monitor.html.erb` dispatcher, added a comment clarifying the `else` branch handles Karambol + Eurokegel + BK2-Kombi, and deleted `_show_bk2_kombi.html.erb` via `git rm`.
