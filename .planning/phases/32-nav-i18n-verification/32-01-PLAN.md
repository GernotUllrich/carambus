**3. rake-tasks-debugging.en.md (3 occurrences, same lines):** Apply the identical fix as the DE version above.

**4. reference/api.de.md (line ~673):** Links to `API.md` which does not exist (the api pages themselves ARE the API reference). Remove the broken link — replace the hyperlinked text with plain text or a self-referential note like "siehe oben" / "see above".
