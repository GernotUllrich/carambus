## BK-2 Game-End-Trigger

| Option | Description | Selected |
|--------|-------------|----------|
| Anstoss-Spieler erreicht Ballziel + Nachstoss-Aufnahme schliesst | Eine Aufnahme nach Anstoss-Ziel | (umgeformt durch User-Korrektur) |
| Innings_goal als hartes Limit | innings_goal=2 für BK-2 | |
| Eigener AASM-State 'nachstoss_active' | Komplexer | |

**User's choice:** User-Korrektur: "Ein BK-2 Spiel mit Nachstoß landet in einer
Totsituation, wenn der Nachstoßende auch das Ballziel erreicht. Der Auslöser für einen
Tiebreak muss in diesem Fall das Erreichen des Ballziels des Nachstoßenden sein."

**Notes:** Trigger ist nicht "eine Aufnahme nach Anstoss-Ziel" sondern "Nachstoss-Spieler
erreicht Ballziel". Wenn Nachstoss-Spieler das Ballziel NICHT erreicht, Game endet mit
A>B (kein Tiebreak). Wenn Nachstoss-Spieler das Ballziel erreicht, Punktgleich → Tiebreak.
