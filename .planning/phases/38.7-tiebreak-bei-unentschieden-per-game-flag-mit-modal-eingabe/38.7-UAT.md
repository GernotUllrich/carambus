Today this case has no automatic detection: an operator who didn't tick the toggle gets no modal.

**Decision (locked, user 2026-04-30):**
> "special case BK-2kombi in BK-2 phase with the special condition on follow up only when player hits balls_goal in first inning and opponent reaches balls_goal as well"

**Required behavior:**

  - `bk2_state` indicates BK-2 phase active

  - Both scores equal at end of set

User's "checkboxen" was colloquial. Hypothesis #4 ruled out — focus on hypotheses #1 (predicate short-circuit), #2 (auto-detect already-resolved guard), and #3 (AASM/reflex submit advance).
