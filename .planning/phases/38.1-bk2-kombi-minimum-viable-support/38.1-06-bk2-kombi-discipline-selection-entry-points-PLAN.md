form, 3-pin credit, D-15 2-point rule, foul negatives, set close, match banner,
i18n) that are all gated on reaching the scoreboard.

`TableMonitorsController#start_game` branch that packs BK2 defaults and CLAMPS
`discipline_a/b` to `"BK2-Kombi"`, a `bk2_options` passthrough in
`TableMonitor::GameSetup#build_result_hash` whitelist, regression coverage for
karambol/snooker/pool, and a system test that reaches the BK2 scoreboard via
navigation (not direct fixture seeding).
