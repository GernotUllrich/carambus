       - For each shot, the scorer should VERIFY:
         - Phase indicator is correct ("Direkter Zweikampf" at match start; "Serienspiel" only if the match reaches that phase per BK2-Kombi rules)
         - Shots-left badge counts down correctly (2 → 1 → turn ends)
         - Bonus shot fires on 5 pins or 5+carom (player stays at table)
         - Foul credits go to the opponent (negative-score side of the engine)
         - Set close fires at 50 points
         - After set close, player_at_table alternates
         - Match close fires at 2 sets won

    3. **Intentional edge-case probes** (scorer deliberately triggers each at least once)
       - A foul with wrong_ball (should credit -6 to opponent)
       - A foul with no_object_ball_hit + band_hit (should credit -1)
       - A foul with no_object_ball_hit + NO band (should credit -2)
       - A bonus shot (5 pins, Direkter Zweikampf)
       - A zero-point shot in Serienspiel (turn should end)
       - Rapid double-tap of submit button (debounce should prevent double-count)

    4. **Issue capture** — For every observed issue (latency, wrong value, UI glitch, confusing label, broken animation, CableReady lag, whatever), log to the Issues Log table in 38.1-05-DRY-RUN-UAT.md with severity (HIGH / MEDIUM / LOW) and a disposition:
