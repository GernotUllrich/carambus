**Canonical shortcut list** (from scoreboard-guide.de.md lines 209-235 and 623-626):
- Arrow Left / Page Up → +1 point, Player A (left)
- Arrow Right / Page Down → +1 point, Player B (right, after player switch)
- `+1` button → +1 point (via button navigation)
- `+5` button → +5 points
- `+10` button → +10 points
- `-1` / `-5` / `-10` → subtract points
- `Next` (DE: Nächster) → player switch
- `Numbers` → open number pad for direct entry (larger scores)
- `1-9, 0` (in number pad) → digits
- `Del` → delete last digit in number pad
- `Esc` → cancel number pad input
- `Enter` → confirm score / activate focused button
- `B` → move right through input buttons (navigation)
- `Down` → activate focused button
- `Protocol` (DE: Protokoll) → open game protocol for inning review/edit
- Timer: `B` → enter timer area, then `Down`/`Enter` → trigger timer action (pause/resume) per scoreboard-guide.de.md lines 629-632
- `^v` / `Ctrl+Z` → undo (documented in scoreboard-guide §Eingabe-Modus; planner may include if space permits — D-03)

The full scoreboard-guide prose is the authoritative source. The card's table is a digest — one row per key/action, with a terse "When" column.
