**D-05: Fix test/controllers/registrations_controller_test.rb**

2. Find the CSRF token extraction that uses a regex like `response.body.match(/<meta name="csrf-token" content="([^"]+)"/)`. Replace with Rails' built-in `assert_select` approach:
   ```ruby
   assert_select 'meta[name="csrf-token"]' do |elements|
     assert_equal 1, elements.size
     token = elements.first["content"]

   end
   ```
   If the extracted token is used in subsequent requests in the same test, keep the extraction but use `css_select` instead of regex:
   ```ruby
   token = css_select('meta[name="csrf-token"]').first["content"]
   ```
   Ensure the test still makes the same assertions about CSRF behavior — just without the brittle regex.
