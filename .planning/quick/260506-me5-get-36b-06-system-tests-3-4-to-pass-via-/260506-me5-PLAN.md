- **Fix:** Replace `assert_includes STARTED_STATES, @tournament.reload.state` with `assert_current_path %r{\A/tournament_monitors/\d+\z}, wait: 10` in Tests 3 and 4.

- **Why this is a real production-edge bug worth filing:** on a fresh local server that hasn't yet synced regions, a system_admin login would hit the same crash in production. Recommended fix: harden the link as `link_to "Migration", migration_cc_region_path(Region[1] || Region.first), class: ...` OR wrap it in `if Region[1].present?`.
