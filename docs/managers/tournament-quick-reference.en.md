# Tournament Quick Reference (Tournament-Day Checklist)

This page is a one-page checklist for tournament day — designed to be printed, laminated, and clipped next to the tournament-day laptop. No explanations, just boxes to tick off. The full step-by-step guide lives in [Tournament Management](tournament-management.md#walkthrough).

<a id="before"></a>
## Before the Tournament Starts (about 2 hours ahead)

- [ ] Laptop powered, internet connection verified
- [ ] Logged into Carambus, tournament visible in the list
- [ ] Tournament synced from ClubCloud ([Step 2](tournament-management.md#step-2-load-clubcloud))
- [ ] Seeding reviewed, participant list complete ([Steps 3 & 4](tournament-management.md#step-4-participants))
- [ ] Participant list finalised ([Step 5](tournament-management.md#step-5-finish-seeding))

    !!! warning "Irreversible click"
        **Finalise participant list** cannot be undone. Only click when the list is truly complete.
        <!-- ref: F-09 -->

- [ ] Tournament mode selected and confirmed ([Step 6](tournament-management.md#step-6-mode-selection))

    !!! tip "Which plan to pick?"
        Carambus usually suggests a plan automatically (e.g. T04 for 5 participants). Accept the suggestion unless you have a specific reason to override it.
        <!-- ref: F-12 -->

- [ ] Start form filled in, parameters checked ([Step 7](tournament-management.md#step-7-start-form))

    !!! tip "English field labels"
        Some parameters currently use English labels or ambiguous wording. When in doubt, accept the defaults.
        <!-- ref: F-14 -->

- [ ] Tables assigned ([Step 8](tournament-management.md#step-8-tables))
- [ ] Scoreboards open on each table's monitor, timer values verified
- [ ] Tournament started ([Step 9](tournament-management.md#step-9-start))

<a id="during"></a>
## During the Tournament

- [ ] Warm-up phase running — players at the tables ([Step 10](tournament-management.md#step-10-warmup))
- [ ] Start released for each match ([Step 11](tournament-management.md#step-11-release-match))
- [ ] Live results being monitored ([Step 12](tournament-management.md#step-12-monitor))
- [ ] Mis-entered scores corrected promptly
- [ ] Absent players noted (cause to be recorded afterwards)
- [ ] When a rule is unclear: call the referee or tournament director

<a id="after"></a>
## After the Final Match

- [ ] All results entered, no open matches ([Step 13](tournament-management.md#step-13-finalize))
- [ ] Tournament finalised (AASM state: closed)
- [ ] Protocol PDFs saved or printed
- [ ] Results uploaded to ClubCloud ([Step 14](tournament-management.md#step-14-upload))

    !!! warning "Wait — do not re-click"
        After clicking upload, the page may look unchanged for several seconds. That is normal — do not click again.
        <!-- ref: F-19 -->

- [ ] Laptop shut down cleanly, power disconnected
