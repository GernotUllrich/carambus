# Turnier-Schnellreferenz (Turniertag-Checkliste)

Diese Seite ist eine einseitige Checkliste für den Turniertag — zum Ausdrucken, Laminieren und Neben-den-Laptop-Klemmen. Keine Erklärungen, nur Haken zum Abarbeiten. Die vollständige Schritt-für-Schritt-Anleitung steht in der [Turnierverwaltung](tournament-management.md#walkthrough).

<a id="before"></a>
## Vor dem Turniertag-Start (ca. 2 Stunden vorher)

- [ ] Laptop an Strom, Internet-Verbindung prüfen
- [ ] In Carambus anmelden, Turnier in der Liste finden
- [ ] Turnier aus ClubCloud synchronisiert ([Schritt 2](tournament-management.md#step-2-load-clubcloud))
- [ ] Setzliste geprüft, Teilnehmerliste vollständig ([Schritt 3 & 4](tournament-management.md#step-4-participants))
- [ ] Teilnehmerliste abgeschlossen ([Schritt 5](tournament-management.md#step-5-finish-seeding))

    !!! warning "Endgültiger Klick"
        **Teilnehmerliste abschließen** ist nicht rückgängig zu machen. Erst klicken, wenn die Liste wirklich vollständig ist.
        <!-- ref: F-09 -->

- [ ] Turniermodus ausgewählt und bestätigt ([Schritt 6](tournament-management.md#step-6-mode-selection))

    !!! tip "Welchen Plan wählen?"
        Carambus schlägt meistens einen Plan automatisch vor (z.B. T04 bei 5 Teilnehmern). Den Vorschlag übernehmen, wenn Sie nicht bewusst eine Alternative wollen.
        <!-- ref: F-12 -->

- [ ] Start-Formular ausgefüllt, Parameter kontrolliert ([Schritt 7](tournament-management.md#step-7-start-form))

    !!! tip "Englische Feldbezeichnungen"
        Einige Parameter heißen derzeit auf Englisch oder sind unklar beschriftet. Im Zweifel die Standardwerte übernehmen.
        <!-- ref: F-14 -->

- [ ] Tische zugeordnet ([Schritt 8](tournament-management.md#step-8-tables))
- [ ] Scoreboards auf jedem Tisch-Monitor geöffnet, Timer-Werte geprüft
- [ ] Turnier gestartet ([Schritt 9](tournament-management.md#step-9-start))

<a id="during"></a>
## Während des Turniers

- [ ] Warmup-Phase läuft — Spieler an den Tischen ([Schritt 10](tournament-management.md#step-10-warmup))
- [ ] Spielbeginn für jede Partie freigegeben ([Schritt 11](tournament-management.md#step-11-release-match))
- [ ] Ergebnisse live mitverfolgen ([Schritt 12](tournament-management.md#step-12-monitor))
- [ ] Falsch eingegebene Scores zeitnah korrigieren
- [ ] Abwesende Spieler notieren (Problem/Ursache später eintragen)
- [ ] Bei Regel-Unklarheiten: Schiedsrichter oder Turnierleitung rufen

<a id="after"></a>
## Nach der letzten Partie

- [ ] Alle Ergebnisse im System eingetragen, keine offenen Partien ([Schritt 13](tournament-management.md#step-13-finalize))
- [ ] Turnier finalisiert (AASM-Status: abgeschlossen)
- [ ] Protokoll-PDFs gespeichert oder gedruckt
- [ ] Ergebnisse nach ClubCloud hochgeladen ([Schritt 14](tournament-management.md#step-14-upload))

    !!! warning "Warten, nicht erneut klicken"
        Nach dem Upload-Klick kann die Seite mehrere Sekunden unverändert aussehen. Das ist normal — nicht erneut klicken.
        <!-- ref: F-19 -->

- [ ] Laptop sauber herunterfahren, Strom trennen
