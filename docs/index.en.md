## ğŸ”§ System Overview

Carambus is a comprehensive billiards tournament management system built with:
- **Backend**: Ruby on Rails 7.2
- **Database**: PostgreSQL
- **Frontend**: Hotwire, Stimulus, Tailwind CSS
- **Real-time**: Action Cable, WebSockets
- **Deployment**: Capistrano
